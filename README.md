# MediBridge

MediBridge is an event-driven healthcare backend platform built using microservices architecture to enable scalable and efficient communication between distributed services.

The project focuses on real-time data streaming, inter-service messaging, and high-performance backend design for healthcare-related workflows.

---

## Project Overview

MediBridge was developed to explore modern backend system design using:

- Event-driven microservices architecture  
- Real-time communication through Kafka  
- High-speed service-to-service interaction using gRPC  
- Containerized deployment for scalability  

This project demonstrates strong experience in building production-style distributed systems.

---

## Key Features

- Event-driven microservices communication  
- Real-time data streaming using Apache Kafka  
- gRPC-based inter-service messaging  
- REST APIs for external integrations  
- Dockerized environment for deployment and scalability  

---

## Tech Stack

- **Backend:** Java, Spring Boot, Microservices  
- **Messaging:** Apache Kafka  
- **Communication:** gRPC  
- **Deployment:** Docker  
- **Architecture:** Event-Driven Distributed Systems
